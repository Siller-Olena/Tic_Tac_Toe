# Tic Tac Toe

## Опис
Проста гра "Тік Так Тое" для двох гравців. Мета гри — заповнити ряд, колонку або діагональ своїми символами (X або O).

## Функціонал
- Створення ігрового поля 3x3
- Ходи двох гравців по черзі
- Перевірка на перемогу або нічию
- Можливість почати гру спочатку

## Технології
HTML-CSS-React

## Встановлення та запуск
1.	Клонувати репозиторій:
git clone https://github.com/Siller-Olena/Tic_Tac_Toe
2. Встановити залежності:
npm install

3. Запустити проект:
npm start
