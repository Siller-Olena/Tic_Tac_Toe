# Product Requirements Document (PRD)

Продукт: Tic Tac Toe
Версія: 1.0
Дата: 21.11.2025
Автори: Губін та Сіллер

## Мета продукту
Створити просту гру "Тік Так Тое", яка дозволяє двом гравцям грати на одному пристрої. Гра повинна відслідковувати перемогу, поразку та нічию.

## Цільова аудиторія
- Школярі та студенти
- Люди, які хочуть швидку настільну гру на комп'ютері

## Функціональні вимоги
1. Ігрове поле 3x3
2. Гравці роблять ходи по черзі
3. Визначення переможця або нічиєї після кожного ходу
4. Можливість перезапуску гри

## Нефункціональні вимоги
- Адаптивний дизайн для різних розмірів екранів
- Легкий інтерфейс
- Без збереження даних після закриття гри

## Сценарії використання
- Гравець А ставить X, гравець B ставить O  
- Перевірка після кожного ходу на перемогу  
- Якщо всі клітини заповнені без переможця — нічия  
- Можливість натиснути "Reset" і почати гру заново

## Обмеження
- Гра тільки для двох гравців на одному пристрої
- Відсутній AI (штучний інтелект)
